# requirements.txt — generador-audio-pruebas
#
# Este programa NO necesita instalar paquetes de Python con pip, NI ffmpeg.
# Usa únicamente la librería estándar de Python 3 (wave, array, csv, subprocess,
# etc.) y la voz que YA TRAE el sistema operativo.
#
# Funciona en:
#
#   macOS    -> usa el comando `say` (integrado en macOS).
#   Windows  -> usa System.Speech / SAPI vía PowerShell (integrado en Windows).
#
# Lo único que necesitas:
#
#   - Python 3.8 o superior.
#       macOS:   ya viene instalado (python3).
#       Windows: instalar desde https://www.python.org/downloads/
#                (marca "Add Python to PATH" durante la instalación).
#
# Para verificar que todo funcione:
#
#   macOS:    ./instalar.sh
#   Windows:  .\instalar.ps1
#
# (No hay líneas de paquetes abajo a propósito: no hace falta ningún pip install.)
